# TG‑Project — Modular RF Tracking Generator

This project builds a modulair Tracking Generator for my HP8562A Spectrum Analyser (22GHz). 
It consists of three RF-modules 

- **Baseband module (0–2.9 GHz)**
- **6 cm module (4.3–7.2 GHz)**
- **3 cm module (8.8–11.7 GHz)**

Each module is completely standalone, with its own PCB, gain budget, LO chain, and SMA interfaces. 
The LOs are centrally generated by a separate PLL/control board.

## Architecture

See `docs/architecture/system-overview.md` for the complete block diagram.

Short summary:

- **HMC219** in the CNC box functions as a downconverter/baseband source.
- **HMC220** is used in the 6 cm and 3 cm upconverter modules.
- LO chains are generated by three PLLs:
- 3.9107 GHz (baseband)
- 4.3 GHz (6 cm)
- 8.8 GHz (3 cm)
- All LOs are filtered with SMA BPFs and routed to the modules via SMA.
